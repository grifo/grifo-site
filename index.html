Produtos

Conheça abaixo os produtos dos quais a Grifo é uma das fundadoras.

  • Eventer

    O Eventer é um sistema voltado para o gerenciamento de eventos, onde o organizador pode controlar todo fluxo financeiro, checklists, participantes, dentre outras funcionalidades como poder gerar um site, acompanhar a hashtag do evento nas redes sociais e muito mais.

  • Moviegames

    Moviegames é uma plataforma de jogos para grandes telas (eventos, cinemas, teatros, festas), onde o cada usuário acessa do seu celular o game que estiver rodando e interage com seu personagem no telão.
